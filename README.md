# Sky anomalies
[Modrinth page](https://modrinth.com/mod/sky-anomalies) 
Is a mod that currently adds meteors that sometimes spawn in the sky.


[Nyoom](https://imgur.com/a/A6xwPXx)

By default these are quite rare (spawning around every half an hour or so), but like most aspects of this mod that is configurable

## Features
### Meteor sizes
Meteor sizes are customizable regular meteors spawn from sizes 1-3 which are non explosive (configruable).

### Configurability
Many aspects of the meteors can be configures including:
- Minimum and maximum spawn size
- Spawn rate
- Explosive-ness

### Sound
As these metors are mostly atmospheric i felt they should have an impact.
If the meteors pass near a player they will hear a flyby sound.
Upon crashing they will release a boom which can be heard from far away (scaling with their size).

### Commands
Meteors can be spawned in creative using the /SpawnMeteor command, and specifying the player around which around which they will spawn, and their size.
Eg.:

```
/spawnMeteor @p 3
```
